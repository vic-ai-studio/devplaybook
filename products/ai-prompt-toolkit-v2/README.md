## How to Use

1. Open the category file that matches your task
2. Find the prompt number (e.g., `DEV-4`, `DW-11`, `BP-3`)
3. Replace all `{{PLACEHOLDER}}` values with your actual content
4. Paste directly into Claude, ChatGPT, or Gemini

### Prompts work best when you:
- Fill in ALL placeholders (remove the `{{` `}}` markers too)
- Add context specific to your situation
- Iterate: use the output as input for a follow-up prompt
