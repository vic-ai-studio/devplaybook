# GitLab CI\n\n## 簡介\nGitLab CI是GitLab內建的持續整合和持續部署工具。\n\n## 基本設定\n```yaml\nstages:\n  - build\n  - test\n  - deploy\n\nbuild_job:\n  stage: build\n  script:\n    - echo \'Building...\'\n    - make build\n\ntest_job:\n  stage: test\n  script:\n    - echo \'Testing...\'\n    - make test\n\ndeploy_job:\n  stage: deploy\n  script:\n    - echo \'Deploying...\'\n    - make deploy\n```\n\n## 最佳實踐\n- 使用cache加速建構\n- 並行化獨立任務\n- 使用artifacts分享建構產物\n- 實施DORA指標追蹤
