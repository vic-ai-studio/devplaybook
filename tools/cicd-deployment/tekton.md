# Tekton\n\n## 簡介\nTekton是Kubernetes原生的建構框架，提供靈活的CI/CD建構區塊。\n\n## 基本設定\n```yaml\napiVersion: tekton.dev/v1beta1\nkind: Pipeline\nmetadata:\n  name: my-pipeline\nspec:\n  tasks:\n    - name: build\n      taskRef:\n        name: build-task\n    - name: test\n      taskRef:\n        name: test-task\n      runAfter:\n        - build\n    - name: deploy\n      taskRef:\n        name: deploy-task\n      runAfter:\n        - test\n\n---\napiVersion: tekton.dev/v1beta1\nkind: Task\nmetadata:\n  name: build-task\nspec:\n  steps:\n    - name: build\n      image: docker:latest\n      script: |\n        #!/bin/sh\n        make build\n```\n\n## 最佳實踐\n- 使用PipelineResources定義輸入輸出\n- 實施TaskResult進行任務間通訊\n- 使用PipelineRun觸發建構\n- 整合Tekton Dashboard監控
