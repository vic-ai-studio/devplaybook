# ArgoCD\n\n## 簡介\nArgoCD是Kubernetes原生的GitOps持續交付工具。\n\n## 基本設定\n```yaml\napiVersion: argoproj.io/v1alpha1\nkind: Application\nmetadata:\n  name: my-app\n  namespace: argocd\nspec:\n  project: default\n  source:\n    repoURL: https://github.com/example/app.git\n    targetRevision: HEAD\n    path: manifests\n  destination:\n    server: https://kubernetes.default.svc\n    namespace: production\n  syncPolicy:\n    automated:\n      prune: true\n      selfHeal: true\n```\n\n## 最佳實踐\n- 實施ApplicationSet大量部署\n- 使用Sync Waves控制部署順序\n- 設定PreSync/PostSync鉤子\n- 整合Rollouts實現進階部署策略
