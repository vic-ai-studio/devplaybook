# Jenkins\n\n## 簡介\nJenkins是開源的自動化伺服器，廣泛用於CI/CD流程。\n\n## 基本設定\n```groovy\npipeline {\n    agent any\n    stages {\n        stage(\'Build\') {\n            steps {\n                sh \'make build\'\n            }\n        }\n        stage(\'Test\') {\n            steps {\n                sh \'make test\'\n            }\n        }\n        stage(\'Deploy\') {\n            steps {\n                sh \'make deploy\'\n            }\n        }\n    }\n}\n```\n\n## 最佳實踐\n- 使用Pipeline即程式碼\n- 實施階段性建構\n- 設定建構超時\n- 使用Docker代理\n\n## 常見外掛\n- Docker Pipeline\n- Git Parameter\n- Pipeline Utility Steps
