# GitHub Actions\n\n## 簡介\nGitHub Actions是GitHub原生的工作流程自動化工具。\n\n## 基本設定\n```yaml\nname: CI/CD Pipeline\n\non: [push, pull_request]\n\njobs:\n  build:\n    runs-on: ubuntu-latest\n    steps:\n      - uses: actions/checkout@v4\n      - name: Set up Python\n        uses: actions/setup-python@v4\n        with:\n          python-version: \'3.11\'\n      - run: pip install -r requirements.txt\n      - run: make build\n\n  test:\n    runs-on: ubuntu-latest\n    needs: build\n    steps:\n      - uses: actions/checkout@v4\n      - run: make test\n```\n\n## 最佳實踐\n- 使用action快取依賴\n- 矩陣策略平行測試\n- 實施工作流程approval\n- 使用environment protection rules
