# CircleCI\n\n## 簡介\nCircleCI是專注於速度的雲端CI/CD平台。\n\n## 基本設定\n```yaml\nversion: 2.1\n\norbs:\n  node: circleci/node@5.0.0\n\njobs:\n  build_and_test:\n    docker:\n      - image: cimg/node:18.10.0\n    steps:\n      - checkout\n      - node/install-packages:\n          app-dir: ~/project\n      - run:\n          name: Build\n          command: npm run build\n      - run:\n          name: Test\n          command: npm test\n\nworkflows:\n  build_and_test_workflow:\n    jobs:\n      - build_and_test\n```\n\n## 最佳實踐\n- 使用Orbs重複使用設定\n- 並行化測試套件\n- 使用SSH Debug進行除錯\n- 實施resource class最佳化
