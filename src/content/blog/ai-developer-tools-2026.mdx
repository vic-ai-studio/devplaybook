## LLM APIs: The Foundation Layer

Before you pick a copilot or assistant, you need to understand the underlying model APIs. These are what power everything else — and if you're building AI features into your own products, you'll be calling these directly.

### Claude API (Anthropic)

Claude is arguably the strongest general-purpose LLM for software development in 2026. Key strengths:

- **200K token context window** (Claude 3.5+ Sonnet) — fits entire codebases in a single prompt
- Exceptional instruction-following: it does what you ask, not what it thinks you meant
- Strong performance on multi-step reasoning tasks (refactoring, architecture decisions)
- Safety features that matter in enterprise contexts

**Pricing (Claude Sonnet 4.6):**
- Input: $3.00 / 1M tokens
- Output: $15.00 / 1M tokens
- Cached input: $0.30 / 1M tokens (huge cost saver for repeated context)

**Best for:** Code generation, large codebase analysis, technical writing, documentation generation.

```javascript
import Anthropic from "@anthropic-ai/sdk";

const client = new Anthropic({ apiKey: process.env.ANTHROPIC_API_KEY });

const response = await client.messages.create({
  model: "claude-sonnet-4-6",
  max_tokens: 2048,
  messages: [
    {
      role: "user",
      content: "Refactor this Express middleware to use async/await and add error handling:\n\n" + code,
    },
  ],
});

console.log(response.content[0].text);
```

### OpenAI API (GPT-4o / o3)

OpenAI's API remains the most widely integrated LLM API, with the largest ecosystem of tooling and the best documentation for beginners.

- **GPT-4o**: Fastest multimodal model — handles text, images, and audio
- **o3/o3-mini**: Best-in-class reasoning model for complex problems
- **Assistants API**: File search, code interpreter, function calling — all managed

**Pricing (GPT-4o):**
- Input: $2.50 / 1M tokens
- Output: $10.00 / 1M tokens

**Pricing (o3-mini):**
- Input: $1.10 / 1M tokens
- Output: $4.40 / 1M tokens

**Best for:** Multimodal apps (image analysis + code gen), production applications needing ecosystem breadth, teams already in the Azure ecosystem.

```python
from openai import OpenAI

client = OpenAI(api_key=os.getenv("OPENAI_API_KEY"))

response = client.chat.completions.create(
    model="gpt-4o",
    messages=[
        {"role": "system", "content": "You are a senior software engineer specializing in Python performance optimization."},
        {"role": "user", "content": f"Profile this code and suggest improvements:\n\n{code}"}
    ]
)

print(response.choices[0].message.content)
```

### Google Gemini API

Gemini's standout feature is its **2M token context window** — the largest available. If you need to feed entire repositories or long documentation corpora into a single prompt, Gemini Flash 2.0 is the most cost-effective way to do it.

**Pricing (Gemini 2.0 Flash):**
- Input: $0.10 / 1M tokens (free tier available)
- Output: $0.40 / 1M tokens

**Pricing (Gemini 1.5 Pro):**
- Input: $1.25 / 1M tokens
- Output: $5.00 / 1M tokens

**Best for:** Bulk code analysis, large document processing, cost-sensitive applications.

```python
import google.generativeai as genai

genai.configure(api_key=os.getenv("GOOGLE_API_KEY"))
model = genai.GenerativeModel("gemini-2.0-flash-exp")

# Feed an entire codebase
with open("my_large_file.py") as f:
    code = f.read()

response = model.generate_content(f"Analyze this codebase and identify security vulnerabilities:\n\n{code}")
print(response.text)
```

## AI Code Copilots: In-Editor Assistance

Copilots live inside your editor. They see your current file, suggest completions, explain errors, and increasingly — they can autonomously make multi-file changes while you supervise.

### Cursor

Cursor is the IDE that has eaten GitHub Copilot's lunch among professional developers. Built on VS Code, it ships with deep AI integration that goes beyond autocomplete:

- **Composer**: Describe a feature, Cursor edits multiple files simultaneously
- **Agent mode**: Autonomous coding with terminal access — runs tests, fixes failures
- **Chat with codebase**: Index your entire repo and ask natural language questions
- **Model selection**: Use GPT-4o, Claude, or Gemini — your choice per session

**Pricing:**
- Free: 2,000 completions + 50 "slow" premium requests/month
- Pro ($20/month): Unlimited completions + 500 fast premium requests
- Business ($40/user/month): Privacy mode, SSO, centralized billing

**Best for:** Full-stack developers who want maximum AI leverage across an entire project. If you're doing complex feature development or large refactors, Cursor's Composer/Agent mode has no equals.

### GitHub Copilot

The OG. Still excellent, especially if you're already in the GitHub ecosystem:

- Deep GitHub integration: PRs, code review suggestions, issue triage
- **Copilot Workspace**: Plan, edit, and PR entire features from an issue description
- Available in VS Code, JetBrains, Neovim, and Visual Studio
- Now powered by GPT-4o and Claude 3.5 Sonnet (model varies by task)

**Pricing:**
- Individual: $10/month (free for students and OSS maintainers)
- Business: $19/user/month
- Enterprise: $39/user/month (includes fine-tuning on your codebase)

**Best for:** Teams standardized on GitHub, developers who want IDE flexibility (JetBrains, Vim), anyone who qualifies for the free tier.

### Codeium (now Windsurf)

Rebranded as **Windsurf** in 2025, Codeium pivoted from pure autocomplete to a full agentic IDE. Its headline feature: **Cascade**, an agent that maintains persistent context across an entire development session.

- Cascade tracks your changes, understands what you're trying to build, and proactively suggests next steps
- **Flow** (autocomplete engine): Trains on your specific codebase patterns
- Strong support for less common languages (Rust, Haskell, Elixir)

**Pricing:**
- Free: Unlimited autocomplete + limited Cascade
- Pro ($15/month): Full Cascade access + unlimited AI flows

**Best for:** Developers who want a fully agentic experience at a lower price point than Cursor, or those working in less mainstream languages.

### Copilot vs Cursor vs Windsurf: When to Choose Each

```
Cursor       → "I want maximum AI power, I'll pay for it"
Copilot      → "I want AI in every IDE I use, I'm in the GitHub ecosystem"
Windsurf     → "I want agentic AI cheaper, I like a clean UX"
```

## Standalone AI Dev Tools

Beyond editors and APIs, there's a growing set of specialized AI tools for specific developer workflows.

### Aider (Terminal-Based AI Coding)

Aider is a CLI tool that runs in your terminal and makes AI-assisted edits directly to your codebase:

```bash
pip install aider-chat
aider --model claude-sonnet-4-6 --file src/api/routes.py

# In the aider prompt:
> Add rate limiting to all POST endpoints using express-rate-limit
```

Unlike editor copilots, Aider works from the terminal, integrates cleanly with git (auto-commits changes), and you can use it with any editor. Great for focused coding sessions or automation scripts.

### Claude Code (Anthropic's CLI Agent)

Claude Code is Anthropic's own CLI coding agent. It operates on your local filesystem, runs terminal commands, reads/writes files, and can complete multi-step tasks autonomously:

```bash
npm install -g @anthropic-ai/claude-code
claude "Add comprehensive error handling to all API endpoints, write tests, and update the README"
```

Unlike most copilots, Claude Code has genuine agentic capabilities — it can run tests, read error output, and iterate until it gets things right.

### Amazon CodeWhisperer (Q Developer)

Now rebranded as **Amazon Q Developer**, this is Amazon's play for enterprise AWS developers:

- Deep AWS service integration: generates CDK, CloudFormation, IAM policies
- Security scanning built-in (OWASP, CVE detection)
- Free tier available within AWS IDE plugins

**Best for:** Teams doing heavy AWS work. The AWS-specific suggestions are genuinely excellent.

## Choosing the Right Stack for Your Workflow

Here's how to think about assembling your AI developer toolkit:

**Solo developer / side projects:**
- Editor: Cursor Free or Windsurf Free
- API: Claude API for complex tasks, Gemini Flash for bulk work
- CLI: Aider for focused coding sessions

**Team of 2–10:**
- Editor: Cursor Pro or GitHub Copilot Business
- API: Claude or OpenAI with caching enabled
- Code review: GitHub Copilot for PRs + custom Claude API prompts

**Enterprise:**
- Editor: GitHub Copilot Enterprise (fine-tuned on your codebase)
- API: Azure OpenAI (compliance) or Anthropic API with business agreement
- Security: Amazon Q Developer for AWS workloads

## Cost Optimization Tips

1. **Enable prompt caching** — Claude and GPT-4 both support caching. If you're repeatedly passing the same system prompt or codebase context, cached tokens cost 80–90% less.

2. **Use smaller models for simple tasks** — Claude Haiku ($0.25/$1.25 per 1M) or GPT-4o-mini ($0.15/$0.60) handle most routine tasks just as well as larger models at a fraction of the cost.

3. **Batch API calls** — OpenAI's Batch API and Anthropic's equivalent offer 50% discounts for non-realtime processing (documentation generation, bulk analysis).

4. **Set max_tokens aggressively** — Most generation tasks don't need 4096 tokens. Setting `max_tokens: 512` for simple tasks prevents runaway costs.

```javascript
// Cost-optimized setup: route by task complexity
function getModel(taskType) {
  const routing = {
    'simple_autocomplete': 'claude-haiku-4-5', // $0.25/1M input
    'code_review': 'claude-sonnet-4-6',         // $3/1M input
    'architecture_design': 'claude-opus-4-6',   // $15/1M input
  };
  return routing[taskType] || 'claude-sonnet-4-6';
}
```
