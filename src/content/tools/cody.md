## What is Cody?

Cody is Sourcegraph's AI coding assistant that combines LLM capabilities with Sourcegraph's powerful code search and intelligence graph. The key differentiator: Cody can reference your *entire codebase* — not just the open files — when generating code or answering questions.

## Key Features

- **Codebase-aware autocomplete**: Suggestions informed by patterns across your whole repo
- **Chat with context**: Ask questions and Cody fetches relevant code from anywhere in your codebase
- **Commands**: Built-in `/edit`, `/explain`, `/test`, `/smell` (find code smells)
- **Multi-repo context**: Enterprise users can query across multiple repos simultaneously
- **Model choice**: Select Claude 3.5 Sonnet, GPT-4o, or Gemini depending on the task

## Free vs Pro

| Feature | Free | Pro ($9/mo) |

## Best For

- Developers working in large monorepos or multi-repo organizations
- Teams already using Sourcegraph for code search
- Enterprise engineering orgs that need audit/compliance controls
- Anyone wanting strong cross-file context at a lower cost than Cursor
